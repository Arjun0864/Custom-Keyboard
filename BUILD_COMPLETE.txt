2. Enable USB Debugging on device
3. Run: adb install ~/Desktop/AB_Keyboard_v1.0.0.apk
4. Wait for installation to complete

METHOD 2: Manual Installation
1. Copy APK to device storage
2. Open file manager on device
3. Tap the APK file

================================================================================
                          SETUP ON DEVICE
================================================================================

STEP 1: Enable Keyboard
Settings → General → Keyboard & input methods → On-screen keyboard → AB Keyboard → Enable

STEP 2: Set as Default
Settings → General → Keyboard & input methods → Default keyboard → AB Keyboard

STEP 3: Launch Settings
Open AB Keyboard app from launcher

STEP 4: Customize
Tap Settings button in keyboard toolbar to customize appearance and behavior

================================================================================
                          TROUBLESHOOTING
================================================================================

INSTALLATION ISSUES:
• If installation fails, try: adb install -r ~/Desktop/AB_Keyboard_v1.0.0.apk
• Make sure USB Debugging is enabled
• Try different USB cable if connection issues

KEYBOARD NOT APPEARING:
• Verify keyboard is enabled in Settings
• Verify keyboard is set as default
• Restart the device
• Reinstall the app

CRASHES:
• Check logcat: adb logcat | grep flutterboard
• Clear app cache: Settings → Apps → AB Keyboard → Storage → Clear Cache
• Reinstall the app
